# Data-Cleaner
Automated data cleaning and transformation utility
## Features
- Lightweight and fast performance
- Cross-platform compatibility
- Simple command-line interface
- Extensible architecture
## Installation
```bash
git clone https://github.com/noobsmoker/data-cleaner-20260507_124659.git
cd data-cleaner-20260507_124659
chmod +x datacleaner.py
```
## Usage
```bash
python datacleaner.py --help
```
## Requirements
- Python 3.6+
- Standard library only
